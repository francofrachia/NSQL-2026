# 🦸 Superheroes SPA - Fullstack MERN

Este proyecto es una Single Page Application (SPA)
Permite gestionar un catálogo de superhéroes de Marvel y DC, permitiendo visualizar detalles, filtrar por casa y realizar operaciones CRUD completas.

## 🚀 Tecnologías Utilizadas

### Backend
* **Node.js & Express**: Servidor y manejo de API REST.
* **MongoDB**: Base de Datos NoSQL.
* **Mongoose**: Modelado de datos y validaciones.
* **Docker**: Contenerización de la base de datos para un entorno de desarrollo consistente.

### Frontend
* **React + Vite**: Interfaz de usuario rápida y moderna.
* **React Router**: Navegación entre secciones (Home, Marvel, DC, Detalles, Formularios).
* **Bootstrap (React-Bootstrap)**: Diseño responsivo y componentes de UI (Cards, Carrusel, Navbar).
* **Axios**: Comunicación con el backend.
